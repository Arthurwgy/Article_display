- [x] 硬币（类似 B 站硬币，通过任务/投稿/签到等渠道获取，每日投币上限 5 枚，对应「热度」打赏）

    1. **上传封面图**（用户从本地上传，或填写图片 URL）
    2. **音乐链接**（粘贴第三方音乐链接，如网易云/QQ 音乐单曲链接）
  - 系统仅存储「封面图 URL + 音乐链接」，不做自动解析
