## 通用规范

| 项 | 规范 |
|---|---|
| 基础路径 | `/api` |
| 认证方式 | JWT，Header `Authorization: Bearer <token>` |
| 数据格式 | `application/json`，请求体与响应体均为 JSON |
| 分页参数 | `?page=N&per_page=M`（默认 per_page=20，最大 100） |
| 时间格式 | ISO 8601 `2026-06-25T14:30:00+08:00` |
| ID 类型 | 字符串 ULID（26 位） |
| 错误响应 | 统一格式 `{ "message": "...", "errors": {...} }` |

### HTTP 状态码

| 码 | 含义 |
|---|---|
| 200 | 成功 |
| 201 | 创建成功 |
| 204 | 删除成功（无响应体） |
| 400 | 参数错误 |
| 401 | 未登录 / token 失效 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 422 | 验证失败 |
| 500 | 服务器错误 |

### 用户角色

| 值 | 名称 | 说明 |
|---|---|---|
| `reader` | 读者 | 默认角色 |
| `author` | 作者 | 发文章，权限同 reader |
| `admin` | 管理员 | 审核 + 后台 |

## P0-1 认证模块

### 1.1 POST `/api/auth/register` 注册

**请求体**
```json
{
  "name": "张三",
  "email": "zhangsan@example.com",
  "password": "password123",
  "password_confirmation": "password123"
}
```

| 字段 | 类型 | 必填 | 校验 |
|---|---|---|---|
| name | string | ✅ | 最长 50 |
| email | string | ✅ | email 格式，未被注册 |
| password | string | ✅ | 最少 8 位 |
| password_confirmation | string | ✅ | 与 password 一致 |

**成功响应 201**
```json
{
  "user": {
    "id": "01HZ...",
    "name": "张三",
    "email": "zhangsan@example.com",
    "role": "reader",
    "avatar": null,
    "bio": null,
    "coin_balance": 0,
    "gold_balance": 0,
    "created_at": "2026-06-25T14:30:00+08:00"
  },
  "token": "eyJ0eXAiOiJKV1Q...",
  "token_type": "bearer",
  "expires_in": 3600
}
```

### 1.2 POST `/api/auth/login` 登录

**请求体**
```json
{ "email": "zhangsan@example.com", "password": "password123" }
```

**成功响应 200**（同注册）

**认证**：需要

**成功响应 204**（无响应体）

## P0-2 文章 CRUD

> 文章状态枚举：`draft`（草稿）/ `pending`（初审中）/ `first_pass`（初审通过）/ `published`（已发布）/ `first_reject`（初审驳回）/ `modify_required`（要求修改）/ `appealing`（申诉二审中）/ `second_pass`（二审通过）/ `second_reject`（最终驳回）

### 2.1 GET `/api/articles` 文章列表

**认证**：可选（付费文章未登录/未付费只返回标题+封面）

**查询参数**

| 参数 | 类型 | 说明 |
|---|---|---|
| page | int | 页码，默认 1 |
| per_page | int | 每页条数，默认 20 |
| category_id | string | 按分类筛选 |
| tag | string | 按标签筛选 |
| author_id | string | 按作者筛选 |
| status | string | 默认只返回 `published` |
| q | string | 搜索标题/作者名 |
| sort | string | `latest` / `hot`（按 view_count） |

**成功响应 200**
```json
{
  "data": [
    {
      "id": "01HZ...",
      "title": "示例文章",
      "slug": "example-article",
      "cover_image": "https://...",
      "is_top": false,
      "is_featured": false,
      "is_paid": true,
      "price_gold": 10,
      "view_count": 100,
      "review_count": 0,
      "published_at": "2026-06-25T14:30:00+08:00",
      "author": {
        "id": "01HZ...",
        "name": "张三",
        "avatar": null
      },
      "category": {
        "id": "01HZ...",
        "name": "技术"
      },
      "tags": ["PHP", "Laravel"]
    }
  ],
  "meta": {
    "current_page": 1,
    "last_page": 5,
    "per_page": 20,
    "total": 100
  }
}
```

### 2.3 POST `/api/articles` 创建文章

**认证**：需要

**请求体**
```json
{
  "title": "新文章",
  "content": "# 标题\n\n正文...",
  "category_id": "01HZ...",
  "tags": ["PHP", "Laravel"],
  "cover_image": "https://...",
  "price_gold": 0,
  "status": "draft"
}
```

| 字段 | 类型 | 必填 | 校验 |
|---|---|---|---|
| title | string | ✅ | 最长 200 |
| content | string | ✅ | Markdown |
| category_id | string | ❌ | 关联分类 |
| tags | string[] | ❌ | 不存在则自动创建 |
| cover_image | string/url | ❌ | 可选 |
| price_gold | int | ❌ | 默认 0，0=免费 |
| status | string | ❌ | 默认 `draft` |

**成功响应 201**（返回完整文章对象）

### 2.4 PUT `/api/articles/{id}` 更新文章

**认证**：需要本人或管理员

**请求体**：同创建，字段可选

**说明**：
- `draft` 状态：可任意修改
- `published` 状态：修改后状态变为 `pending`，重新走审核

**成功响应 200**

### 2.5 DELETE `/api/articles/{id}` 删除文章

**认证**：需要本人或管理员

**说明**：仅 `draft` 状态可删除，其他状态返回 403。

**成功响应 204**

### 2.6 POST `/api/articles/{id}/submit` 提交审核

**认证**：需要本人

**说明**：`draft` → `pending`（先走机审，再进初审队列）

**成功响应 200**

## P0-3 审核流程

### 3.1 GET `/api/articles/{id}/review-logs` 审核日志

**认证**：需要本人或管理员

**响应 200**
```json
{
  "data": [
    {
      "id": "01HZ...",
      "action": "first_pass",
      "reason": null,
      "review_round": 1,
      "reviewer": {
        "id": "01HZ...",
        "name": "管理员"
      },
      "created_at": "2026-06-25T14:30:00+08:00"
    }
  ]
}
```

**action 枚举**：
`auto_reject` / `first_pass` / `first_reject` / `modify_required` / `second_pass` / `second_reject`

### 3.2 POST `/api/articles/{id}/appeal` 申诉

**认证**：需要本人

**说明**：`first_reject` 状态才能申诉，提交后状态变为 `appealing`。

**请求体**
```json
{ "reason": "我认为这是误判" }
```

**成功响应 200**

### 3.3 POST `/api/articles/{id}/resubmit` 修改后重新提交

**认证**：需要本人

**说明**：`modify_required` 状态重新提交，机审后进入初审队列。

**成功响应 200**
